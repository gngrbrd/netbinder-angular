Substitution Variables
&VariableName
  • &VaribleName
  • DEFINE and UNDEFINE
  • &&VaribleName

Use substitution variables to create dynamic queries by replacing comlum names or values with varibles. Enclose variables referencing string values in single quotes.


SELECT column_name = &ColumnVariable
FROM table_name = &TableVariable
WHERE column_name = '&Value'
DEFINE and UNDEFINE

Use DEFINE to create variable for session. Use UNDEFINE to variable for session.


DEFINE Value = 200

SELECT column_name = &ColumnVariable
FROM table_name = &TableVariable
WHERE column_name = &Value

UNDEFINE Value
