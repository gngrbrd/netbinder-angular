Substitution Variables
&VariableName
  • &VaribleName
  • DEFINE and UNDEFINE
  • ACCEPT and PROMPT
  • &&VaribleName

Use substitution variables to create dynamic queries by replacing comlum names or values with varibles. Enclose variables referencing string values in single quotes.


SELECT column_name = &ColumnVariable
FROM table_name = &TableVariable
WHERE column_name = '&Value'
ACCEPT and PROMPT

Using substitution variable will display a prompt window to indicate values to be entered. ACCEPT and PROMP allows customizing the prompt window. 


ACCEPT emp_from PROMPT 'please enter EMPLOYEE from ' ;
ACCEPT emp_to PROMPT 'please enter EMPLOYEE to ' ;
SELECT * FROM EMPLOYEES
WHERE EMPLOYEE_id between &emp_from and &emp_to;
DEFINE and UNDEFINE

Use DEFINE to create variable for session. Use UNDEFINE to destroy session variable.


DEFINE Value = 200

SELECT column_name = &ColumnVariable
FROM table_name = &TableVariable
WHERE column_name = &Value

UNDEFINE Value
&&VariableName

Use double ampersand to create variable for session and assign it at the same time. Use UNDEFINE to destroy session variable.


SELECT column_name = &ColumnVariable
FROM table_name = &TableVariable
WHERE column_name = &&Value

UNDEFINE Value
